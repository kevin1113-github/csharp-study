# **csharp-study**

## [C# 프로그래밍 입문](https://wikidocs.net/book/14086)

- 해당 교재로 4주 동안 C# 언어 기초를 공부합니다.
- 실습은 매주 선생님과 함께 진행하며, 과제는 스스로의 힘으로 할 수 있는 만큼 도전해봅니다.
- C#의 기본 문법부터 객체지향 프로그래밍, 최신 문법까지 차근차근 배워봅시다!

## 주차별 학습 목표

### 1주차: 프로그래밍 기본기 다지기 (기본 문법)

- **학습 목표:**
  - GitHub, VisualStudio, VSCode, .NET SDK 등 C# 개발환경 설정
  - 변수, 자료형, 연산자, 제어문 (조건문, 반복문), 입출력 등 C#의 기본 문법 익히기
- **실습: 구구단 콘솔 애플리케이션 만들기**
  - 원하는 단(예: 3단)을 입력하면 3 x 1 = 3부터 3 x 9 = 27까지 화면에 예쁘게 출력하는 프로그램을 만듭니다.
- **과제: 숫자 맞추기 게임 (Up & Down)**
  - 컴퓨터가 1부터 100 사이의 비밀 숫자를 무작위로 정합니다. 학생이 숫자를 입력하면 컴퓨터가 "더 높아요(Up)!" 또는 "더 낮아요(Down)!"라고 힌트를 주며 맞추는 게임입니다.
  - 랜덤 숫자 생성 함수를 배워서 진행합니다. (예: Random.Next(1, 101))

### 2주차: 코드의 구조화와 데이터 관리 (함수와 컬렉션)

- **학습 목표:**
  - 함수(메서드) 정의와 호출, 매개변수와 반환값 이해하기
  - 배열과 리스트 등 컬렉션 자료구조 익히기
  - 파일입출력을 이용한 데이터 저장과 불러오기 방법 배우기
- **실습: 콘솔 자판기 돈 계산기**
  - 콜라, 사이다, 환타의 가격을 리스트(또는 배열)에 저장해 둡니다. 메뉴를 선택하고 돈을 넣으면 거스름돈을 계산해 주는 함수를 만듭니다. (예: CalculateChange(int inputMoney, int itemPrice))
- **과제: 나만의 아이돌/게임 캐릭터 도감 만들기**
  - 좋아하는 아이돌 멤버 이름이나 게임 캐릭터 이름을 리스트에 저장하고 관리하는 프로그램입니다.
  - 파일 입출력을 사용하여 새로운 이름을 추가(Add)하거나, 전체 목록을 보거나, 특정 이름을 검색하는 기능을 구현합니다.

### 3주차: C#의 핵심, 객체지향 프로그래밍 (OOP)

- **학습 목표:**
  - 클래스와 객체의 개념 이해하기
  - 캡슐화, 상속, 다형성 등 OOP의 4대 원칙 익히기
- **실습: RPG 게임 캐릭터 설계도 그리기 (클래스와 객체)**
  - Character 클래스를 만들고, 체력(HP), 공격력(ATK), 이름(Name)이라는 변수(필드)를 줍니다. 그리고 "공격하기", "물약 마시기"라는 함수(메서드)를 넣습니다. 이 설계도로 '전사' 객체와 '마법사' 객체를 직접 찍어내어 서로 싸우게 해봅니다.
- **과제: 몬스터 키우기 (상속과 다형성)**
  - 기본 Monster 클래스를 상속받아 Slime(슬라임)과 Dragon(드래곤)을 만듭니다.
  - 모든 몬스터는 "울부짖기"라는 공격 기술이 있지만, 슬라임은 "말랑!"하고 울고, 드래곤은 "크아앙!"하고 울도록 다형성(오버라이딩)을 적용해 봅니다.

### 4주차: 현대적인 C# 문법 익히기 및 미니 프로젝트

- **학습 목표:**
  - 람다식, LINQ, 비동기 프로그래밍 등 최신 C# 문법 익히기
- **실습: LINQ로 마트 영수증 필터링하기**
  - 장바구니 물품 목록 리스트에서 LINQ를 사용해 "5,000원 이상인 물건만 골라내기", "가격 순으로 정렬하기"를 단 한 줄의 코드로 구현해 봅니다. 람다식의 편리함을 맛보는 실습입니다.
- **과제: [미니 프로젝트] 텍스트 기반 타이쿤 (카페/편의점 경영 게임)**
  - 내용: 1~3주차에 배운 모든 개념(반복문, 리스트, 클래스)을 총동원합니다. 매 턴마다 메뉴를 개발하고, 손님에게 음료를 팔아 돈을 버는 간단한 텍스트 경영 게임입니다. (비동기 프로그래밍 async/await을 살짝 얹어서 "3초 동안 커피 추출 중..." 처럼 대기 시간을 주는 연출을 넣어도 좋습니다.)
